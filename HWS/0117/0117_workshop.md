1. 세로로 출력하기

   ```python
   for i in range(1, int(input("number"))+1):
       print(i)
   ```

2. 거꾸로 세로로 출력하기

   ```python
   for i in range(int(input("num")), 0, -1):
       print(i)
   ```

3. N줄 덧셈

   ```python
   a = int(input("num"))
   sum = 0
   for i in range(1, a+1):
       sum += i
   print(sum)
   ```
